Personal Projects

Spell Checker

This program reads an essay/file from the user, and checks the spelling of each word. For each incorrect word, it offers suggested corrections. After each correction, the final essay is written to a new file. The full dictionary is stored in a hash table that uses separate chaining with linked lists. Written in Java.

Bash Shell

Implementation of a unix shell. Includes shell built-ins, process creation, background jobs, redirection, and batch jobs. Written in c.

Simple Web Server

A straightforward web server that handles get and post requests and processes forms. It uses the select feature to handle multiple requests at once. Written in python.

Merkle Tree String Verifier

An initial plunge into functional programming! A functional implementation of a merkle tree. The program uses the merkle tree to test whether certain strings/words are included in a sentence or not. I wrote two separate versions of this - one in haskell and one in ocaml.

Travelling Salesman

An implementation and analysis of Kruskal's algorithm, Boruvka's algorithm, and two different approximation algorithms for the travelling salesman's problem. Originally used mileage data from the Stanford Graph Base.

Ternary Logic Simulator

A discrete event simulation of a ternary logic circuit. Uses inheritence to build-out different kinds of circuits. Written in Java.

Google Maps Neighborhood Map

This project pulls in 10 art, science and history museums in the Washington D.C. area from the Foursquare API, and pulls in a corresponding wikipedia article for each location. The museum's location appears on a google map, and its name, address and relevant wikipedia article are included in a pop-up window and listed in the sidebar. The user can filter the list of museums in the sidebar by using the category filter (implemented via knockout.js)

Catalog Application

This project displays items by category. The home page displays all of the categories in the left column, and all of the items in the right column. The user has the option to add a new item by clicking the "Add Item" link near the top of the right column. When the user clicks on a category, the center column only shows the items associated with the chosen category. Alternatively, when the user clicks on an item, the page only shows the content associated with the chosen item. If the user authored the item in question, then edit and delete links will appear below the item's content. The application uses google plus and facebook for oauth2 authentication.

Logs Analysis

This project reports three different statistics from the a news database. First, it prints out the top three articles by number of views. Second, it outputs the top three authors by number of views. Third, it prints out days on which more than one percent of requests led to errors. For each statistic, there are two corresponding functions, one that fetches the records from the database, and another that prints out the results in an appropriate, formatted manner. Finally, a main function runs all three output functions.

Movie Trailer Site
